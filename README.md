# 🎵 Projeto Music Hope

Projeto de Web Design focado no estudo do framework **Bootstrap 5.3**. O site explora a evolução da música, da Era Disco às batidas contemporâneas de 2026.

## 🛠️ Tecnologias Utilizadas
* HTML5 / CSS3
* [Bootstrap 5.3](https://getbootstrap.com/)
* Bootstrap Icons

## 📋 Requisitos Implementados (Checklist)
1. **Grid System:** Uso de `.container`, `.row` e `.col` para layout responsivo.
2. **Cards Customizados:** Seção de destaques com efeitos de sombra e cores da paleta.
3. **Accordion:** Seção de curiosidades musicais com efeito colapsável.
4. **Formulário de Contato:** Inclui validação via JavaScript nativo do Bootstrap.
5. **Componentes Extras (Item 10):** - **Input Groups:** Formatação avançada no campo de e-mail.
   - **Modais:** Confirmação de envio de formulário para melhor UX.
